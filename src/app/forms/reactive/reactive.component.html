• step 1: copy or create form structure
  • step 2: for reactive Form , we don't need FormsModule to be imported in AppModule we can remove that , instead of that we have to import ReactiveFormsModule in AppModule
  • step 3: setup Forms and Controls :

    1. first we need to declaire property in .ts file of type FormGroup ex: myReacticeForm: FormGroup

    2. after that in ngOninit lifecycle hook we have to setup or initilise the form controls as fallow

    this.myReacticeForm = new FormGroup({ 'username': new FormControl(null), 'email': new FormControl(null), 'gender': new FormControl("Male"), })

    3. then connect the Html form with property in ts file here we have to use the directive [FormGroup] on form tag on html and bind that with the form property on .ts file . [FormGroup]="myReacticeForm"

    4. till now control is not bind so to bind that we have to use FormControlName directive in each input element and bind this directive with control we declaire in myReacticeForm in .ts file

    5. for submitting form we will use same ngSubmit directive and here we don't need to add local template refference varibale
