ViewChild , Renderer and ContentChild

we can modify and manupulate DOM element with the help of view child decortor and we also can do the same with child Component element from the parent Component. But this is not correct way to access the DOM element.

Angular has provide the better way to access and manupulate DOM element is by Using Renderer. Renderer is Angular inbuilt class we use it as service in angular and using which we can do manupulation in DOM.

if we using child component and projecting content using ng-content in child component then to access and modify that ng-content we can use ContentChild decortor
