HOSTLISTNER and HOSTBINDING are concept of Directives

HOST element : the element on which we apply our custom directive that element is Host element of that directive. That Hosts our directive so that is Host element

HOSTLISTNER

To capture the event of Host element we use HostListner. Host Listner help to "Listen and Handle event on Host Element" here we can find any event of host element and we can bind custom logic to it .

Host Listner use at Component
Host Listner use at Directive

HOSTBINDING

it is used to bind property on host element. instead of using renderer we can use Host binding it's alernative way for it.
